To run the Back-end server (REST API - Implement using nodeJs)

    -> Open the file in IDE(VS code, Intelj).
    -> run 'npm install' command in the terminal to download the npm modules.
    -> To start the server, in the terminal type and run 'npm start'
    -> Then, open the browser and type on URL as 'http://localhost:5000/'
